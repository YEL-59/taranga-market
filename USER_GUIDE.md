## üéØ Using Filters

### Desktop Filter Sidebar
Located on the left side of category pages:

#### Common Filters Across Categories:
1. **Location/City**: Filter by city (Dakar, Thies, Saint-Louis, etc.)
2. **Price Range**: 
   - Minimum Price input
   - Maximum Price input
3. **Reset Filters**: Clear all active filters

#### Category-Specific Filters:

**Vehicles:**
- Make (Toyota, Mercedes, BMW, etc.)
- Model
- Year
- Mileage range
- Transmission type
- Fuel type

**Properties:**
- Property type (For Rent, For Sale)
- Number of bedrooms
- Amenities (WiFi, Pool, Parking, etc.)

**Products/Phones:**
- Category (Phones, Computers, Tablets, Accessories)
- Condition (New, Used, Refurbished)

**Services:**
- Service category (Plumbing, Cleaning, Electrical, etc.)

**Jobs:**
- Job type (Full-time, Part-time, Contract, Freelance)
- Experience level (Entry, Junior, Mid-level, Senior)
- Industry
- Remote options (Yes, No, Hybrid)

### Mobile Filters
1. Tap the **"Filters"** button (top-right with slider icon)
2. Filter panel slides in from the left
3. Select your filter criteria
4. Tap **"Show X Results"** to apply filters
5. Panel closes automatically

### Applying Filters
1. Select your desired filter options
2. Results update automatically
3. Active filters are highlighted
4. Use "Reset Filters" to clear all selections

## ‚ù§Ô∏è Managing Favorites

### Adding to Favorites
1. Click the **heart icon** on any item card
2. Icon fills with color when added
3. Item is saved to your favorites list

### Viewing Favorites
1. Click the **heart icon** in the top navigation bar
2. Opens the Favorites page
3. View all your saved items in one place

### Removing from Favorites
1. Click the **filled heart icon** on any favorited item
2. Icon becomes outlined (unfilled)
3. Item is removed from favorites

### Favorites Page Features
- Grid layout of all saved items
- Same card design as category pages
- Quick access to view details
- Remove items directly from this page
- Empty state message if no favorites

## üí° Tips & Best Practices

### For Buyers/Browsers

1. **Use Filters Wisely**
   - Start broad, then narrow down
   - Combine multiple filters for precise results
   - Reset filters if you get zero results

2. **Save Your Favorites**
   - Heart items you're interested in
   - Compare multiple options later
   - Don't lose track of good deals

3. **Check Details Carefully**
   - Read full descriptions
   - View all available photos
   - Note the location and seller info

4. **Contact Sellers**
   - Use provided contact methods
   - Ask questions before committing
   - Verify item condition and availability

### For Optimal Experience

1. **Internet Connection**
   - Use WiFi for best image quality
   - Mobile data works but may load slower

2. **Browser Compatibility**
   - Use latest browser versions
   - Enable JavaScript
   - Allow cookies for favorites to persist

3. **Screen Size**
   - Desktop: Best for detailed browsing and filtering
   - Tablet: Great balance of portability and screen space
   - Mobile: Perfect for quick browsing on-the-go

### Search Strategy

1. **Start with Categories**
   - Go directly to relevant category
   - Use "All Items" for cross-category browsing

2. **Apply Location Filters First**
   - Find items near you
   - Reduces shipping/travel needs

3. **Set Price Range**
   - Define your budget early
   - Avoid wasting time on out-of-range items

4. **Sort and Compare**
   - Use favorites to create a shortlist
   - Compare specs side-by-side
